# Movies
Представляет из себя приложение, в котором можно посмотреть трейлеры, комментарии, обложки и критики (отзывы) по фильму. Также есть возможность "сохранить в избранном" всю информацию о фильме в база данных, она будет доступна также в офлайне. 

Стек: RxJava, Room, Retrofit, Gson, Glide
